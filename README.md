# 五胡乱华时期势力变化地图 (304-439 AD)

交互式历史地图，展示中国五胡十六国时期（304-439年）各政权势力范围的动态变迁。

## 功能特点

- 🗺️ **交互式地图** — 基于 Leaflet.js，展示各政权彩色势力范围
- ⏱️ **时间滑块** — 拖动底栏或使用键盘方向键在 304-439 年之间切换
- ▶️ **自动播放** — 一键播放势力演变动画，可调速度 (0.5x ~ 10x)
- 🖱️ **悬停提示** — 鼠标悬停领土或都城显示政权详情（名称、民族、统治者）
- 🔍 **点击聚焦** — 点击图例或领土自动放大聚焦到对应政权
- 🏷️ **都城标注** — 各政权都城以标记点显示
- 🌊 **地理参考** — 标注黄河、长江、淮河走向
- 📋 **历史事件** — 右侧面板显示每年关键历史事件
- 📱 **响应式设计** — 支持桌面和移动端
- ⌨️ **全键盘操作** — 方向键切换年份，空格播放/暂停

## 操作说明

| 操作 | 方式 |
|------|------|
| 切换年份 | 拖动底部滑块 / ← → 方向键 |
| 快进/快退 5 年 | ↑ ↓ 方向键 |
| 播放/暂停 | ▶ 按钮 / 空格键 |
| 切换速度 | 点击速度标签 (1x / 2x / 3x / 5x / 10x) |
| 缩放地图 | 鼠标滚轮 (地图空白处) / Ctrl+滚轮 |
| 查看详情 | 悬停在政权领土或都城标记上 |
| 聚焦政权 | 点击图例条目或政权领土 |
| 关键节点跳转 | 底部快捷年份按钮 |

## 使用方法

直接浏览器打开 `index.html`，无需安装或构建。

> ⚠️ 地图使用 CartoDB 瓦片服务，在中国大陆可能需要 VPN 才能加载底图。

## 数据说明

领土边界基于以下资料近似绘制：
- 《中国历史地图集》（谭其骧主编）
- 《十六国春秋》《晋书》《资治通鉴》等正史
- 现代学术研究对十六国疆域的考证

> 📝 古代疆域边界为近似值，各政权边界在实际历史中更为复杂且动态变化。本地图将疆域简化为多边形以展示整体变迁趋势，精确学术边界请参考专业历史地理学著作。

## 技术栈

- [Leaflet.js](https://leafletjs.com/) — 开源地图库
- CartoDB Positron — 浅色底图瓦片
- 纯原生 HTML/CSS/JavaScript，无框架依赖
